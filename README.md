# Web3 Glossary

A simple, beginner-friendly website that explains common Web3 terms in plain language.

## Features

- Clean, modern design
- Responsive layout for mobile and desktop
- Easy-to-understand explanations of Web3 terminology
- No JavaScript required - pure HTML and CSS

## Local Development

1. Clone this repository
2. Open `index.html` in your web browser
3. That's it! No build process required

## Deployment Options

This static website can be deployed to any web hosting service. Here are some popular options:

### Netlify

1. Push your code to GitHub
2. Connect your GitHub repo to Netlify
3. Deploy with one click

### Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts

### GitHub Pages

1. Push your code to GitHub
2. Go to Settings > Pages
3. Select source branch and save

### Surge.sh

1. Install Surge: `npm i -g surge`
2. Run `surge` in the project directory
3. Follow the prompts
